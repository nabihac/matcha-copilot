# 🍵 Matcha Cart: Pop-Up AI Co-Pilot

An interactive internal operations and logistics dashboard built using **Python**, **Streamlit**, and the **Google GenAI SDK** to optimize inventory demand constraints and supply chains for a high-energy beverage pop-up venture.

## 🚀 Project Overview
<img width="1470" height="834" alt="Screenshot 2026-06-12 at 3 49 05 PM" src="https://github.com/user-attachments/assets/55d6e88c-15c3-41b3-bb0d-09b651922412" />

This tool eliminates manual supply-ordering guesswork by integrating rule-based baseline logic with localized LLM processing. The app calculates standard ingredient constraints (grams of matcha, milk cartons, and cups needed) based on adjustable foot traffic parameters, dynamically discounts demand based on inclement weather variables, and leverages the **Gemini 2.5 Flash** model to output custom operational and marketing briefs.

## 🛠️ Built With
* **Python 3.13** - Core application logic
* **Streamlit** - Front-end dashboard and UI architecture
* **Google GenAI SDK** - API routing for generative intelligence summaries
* **Statsmodels / Pandas** - Data formatting and mathematical array operations
